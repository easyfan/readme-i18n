# readme-i18n

**Генерирует, обновляет, переводит и удаляет** локализованные файлы README для любого
каталога. Каждая языковая версия пишется нативно — без машинного перевода с английского.

Набор языков по умолчанию: **zh · en · de · fr · ru** (5 из 6 официальных языков ООН,
~4 млрд носителей)

---

## Возможности

- **Генерация**: определить содержимое из исходного кода и создать несколько языковых версий
- **Обновление**: применить изменения ко всем языковым версиям одновременно
- **Перевод**: расширить одноязычный README до полного набора языков
- **Удаление**: удалить все версии или конкретный язык с подтверждением

Активация на естественном языке:

```
Добавь README в packer/my-plugin
Создай многоязычный README для этого проекта
Обнови только немецкий README с учётом новых изменений API
Удали русский README
```

## Установка

### Вариант А — плагин Claude Code (рекомендуется)

```
/plugin marketplace add easyfan/readme-i18n
/plugin install readme-i18n@readme-i18n
```

### Вариант Б — скрипт установки

```bash
git clone https://github.com/easyfan/readme-i18n
cd readme-i18n
./install.sh
```

```bash
# Параметры
./install.sh --dry-run      # предварительный просмотр без записи файлов
./install.sh --uninstall    # удалить установленные файлы
CLAUDE_DIR=/custom ./install.sh
```

### Вариант В — ручная установка

```bash
cp -r skills/readme-i18n ~/.claude/skills/
```

## Поддержка языков

Принимаются стандартные коды BCP-47 и распространённые псевдонимы. Суффикс имени файла соответствует введённому коду:

| Ввод | Язык содержимого | Выходной файл |
|------|-----------------|---------------|
| `cn`, `zh-cn`, `zh-hans` | Упрощённый китайский | `README-cn.md` / `README-zh-cn.md` … |
| `zh`, `zh-sg` | Упрощённый китайский | `README-zh.md` |
| `zh-tw`, `zh-hk`, `zh-hant`, `zht` | Традиционный китайский | `README-zh-tw.md` … |
| `en`, `en-us`, `en-gb` | Английский | `README.md` |
| `de`, `de-at`, `de-ch` | Немецкий | `README-de.md` … |
| `fr`, `fr-ca`, `fr-be` | Французский | `README-fr.md` … |
| `ru` | Русский | `README-ru.md` |
| Любой другой BCP-47 | Соответствующий язык | `README-<код>.md` |

Недопустимые коды отклоняются с понятным сообщением. `cn` распознаётся как упрощённый китайский.

> **Примечание о наборе по умолчанию**: zh · en · de · fr · ru охватывает 5 официальных языков ООН.
> Добавьте `es` (испанский) или `ja` (японский) для расширения охвата.

## Создаваемые файлы

| Режим | Язык | Имя файла |
|-------|------|-----------|
| Один язык | Английский (en) | `README.md` |
| Один язык | Любой другой | `README-<код>.md` — `README.md` не создаётся |
| Несколько языков | Английский (en) | `README.md` |
| Несколько языков | Назначенный основной (не английский) | `README.md` |
| Несколько языков | Все остальные | `README-<код>.md` |

В многоязычном режиме в начало каждого файла вставляется навигационная строка:
```markdown
[English](README.md) | [中文](README-zh.md) | [Deutsch](README-de.md) | ...
```

При перезаписи 2 файлов или более предлагается групповое подтверждение:
```
Уже существует 3 файла: README.md  README-zh.md  README-de.md
Перезаписать всё? [Y] Все / [N] Пропустить все / [D] Решать по каждому
```

**Изменения и удаления по умолчанию применяются ко всем языковым версиям.** Для конкретного языка укажите его явно.
