# readme-i18n

Генерирует локализованные файлы README для любого каталога — на одном языке или сразу на нескольких.

Набор языков по умолчанию: **zh · en · de · fr · ru**

---

## Установка

### Вариант А — плагин Claude Code (рекомендуется)

```
/plugin marketplace add easyfan/readme-i18n
/plugin install readme-i18n@readme-i18n
```

### Вариант Б — скрипт установки

```bash
git clone https://github.com/easyfan/readme-i18n
cd readme-i18n
./install.sh
```

```bash
# Параметры
./install.sh --dry-run      # предварительный просмотр без записи файлов
./install.sh --uninstall    # удалить установленные файлы
CLAUDE_DIR=/custom ./install.sh
```

### Вариант В — ручная установка

```bash
cp -r skills/readme-i18n ~/.claude/skills/
```

## Использование

Попросите Claude создать README для нужного каталога:

```
Добавь README в packer/my-plugin
Создай многоязычный README для этого проекта
Создай README для src/utils — только на китайском и английском
```

После активации skill сначала проверяет наличие файлов README в целевом каталоге. Если они
найдены, предлагается обновить существующие, добавить недостающие языки или пересоздать всё
с нуля. В противном случае задаётся вопрос о режиме языка:

```
Создать README:
  1. На одном языке
  2. На нескольких языках  [по умолчанию: zh · en · de · fr · ru]
```

- **Один язык**: выберите из `zh / cn / en / de / fr / ru` или введите любой код BCP-47
- **Несколько языков**: подтвердите набор по умолчанию или настройте его

Если выбранный набор не включает английский, перед генерацией указывается один язык в
качестве основной версии (`README.md`).

## Псевдонимы языков

Поддерживаются распространённые псевдонимы; суффикс имени файла соответствует вводу пользователя:

| Ввод | Язык содержимого | Выходной файл |
|------|-----------------|---------------|
| `cn`, `zh-cn`, `zh-hans` | Упрощённый китайский | `README-cn.md` / `README-zh-cn.md` … |
| `zh`, `zh-sg` | Упрощённый китайский | `README-zh.md` |
| `zh-tw`, `zh-hk`, `zh-hant`, `zht` | Традиционный китайский | `README-zh-tw.md` … |
| `en`, `en-us`, `en-gb` | Английский | `README.md` |
| Любой другой BCP-47 | Соответствующий язык | `README-<код>.md` |

Недопустимые коды (например, `xyz123`) отклоняются с понятным сообщением об ошибке.

## Создаваемые файлы

| Режим | Язык | Имя файла |
|-------|------|-----------|
| Один язык | Английский (en) | `README.md` |
| Один язык | Любой другой | `README-<код>.md` |
| Несколько языков | Английский (en) | `README.md` |
| Несколько языков | Назначенный основной (не английский) | `README.md` |
| Несколько языков | Все остальные | `README-<код>.md` |

Если перезапись затронет 2 файла или более, предлагается групповое подтверждение:
```
Уже существует 3 файла: README.md  README-zh.md  README-de.md
Перезаписать всё? [Y] Все / [N] Пропустить все / [D] Решать по каждому
```

**Изменения и удаления по умолчанию применяются ко всем языковым версиям.** Чтобы затронуть только конкретный язык, укажите его явно.
