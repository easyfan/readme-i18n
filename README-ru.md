# readme-i18n

Генерирует локализованные файлы README для любого каталога — на одном языке или сразу на нескольких.

Набор языков по умолчанию: **zh · en · de · fr · ru**

---

## Установка

### Вариант А — плагин Claude Code (рекомендуется)

```
/plugin marketplace add easyfan/readme-i18n
/plugin install readme-i18n@readme-i18n
```

### Вариант Б — скрипт установки

```bash
git clone https://github.com/easyfan/readme-i18n
cd readme-i18n
./install.sh
```

```bash
# Параметры
./install.sh --dry-run      # предварительный просмотр без записи файлов
./install.sh --uninstall    # удалить установленные файлы
CLAUDE_DIR=/custom ./install.sh
```

### Вариант В — ручная установка

```bash
cp -r skills/readme-i18n ~/.claude/skills/
```

## Использование

Попросите Claude создать README для нужного каталога:

```
Добавь README в packer/my-plugin
Создай многоязычный README для этого проекта
Создай README для src/utils — только на китайском и английском
```

После активации skill задаёт вопрос:

```
Создать README:
  1. На одном языке
  2. На нескольких языках  [по умолчанию: zh · en · de · fr · ru]
```

- **Один язык**: выберите из zh / en / de / fr / ru или введите любой код BCP-47
- **Несколько языков**: подтвердите набор по умолчанию или настройте его

---

## Создаваемые файлы

| Язык | Имя файла |
|------|-----------|
| Английский (en) | `README.md` |
| Китайский (zh) | `README-zh.md` |
| Немецкий (de) | `README-de.md` |
| Французский (fr) | `README-fr.md` |
| Русский (ru) | `README-ru.md` |
| Другой | `README-<код>.md` |

В многоязычном режиме `README.md` всегда соответствует английской версии (или назначенному основному языку, если английский не выбран).

**Изменения и удаления по умолчанию применяются ко всем языковым версиям.** Чтобы затронуть только конкретный язык, укажите его явно.
